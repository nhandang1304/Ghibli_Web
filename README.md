## `Feature`:
- Displays Studio Ghibli films 
- Details page (content, director, date, rating, trailer,..)
- Search functionality for films
- Display a horizontal sliding carousel in Home page

## `Technologies Used`

### I. Frontend
- HTML
- CSS
- Boostrap
- JavaScript

###	II. Backend
- ASP.NET / ASP.NET Core (C#)

### III. Database
- SQL Server

## `How to run Project`

### 1. Clone the repository

`git clone https://github.com/nhandang1304/Ghibli_Web.git`

### 2. Open the project in Visual Studio

### 3. Database Setup Instructions
- From menu, select <b>View → Server Explorer</b>, then click <b>Data Connections -> Add Connection </b>
- Select <b>Microsoft SQL Server Database File</b>, then browse to <b>Database1.mdf<b> in the folder <b>App_Data</b>
### 4. Run the application using IIS Express or dotnet run
